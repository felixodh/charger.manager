# `{charger.manager}`

This package serves as a tool to inspect the potential installation
sites for electric vehicle chargers.

It:

- gathers data sources (chargers, infrastructure, population data)
- extracts, loads, and transforms these data
- visualizes the data (under construction)
- proposes additional charger potential for public and private
  stakeholders (under construction)
